# Career Compass AI

## Overview
Career Compass AI is a smart web application designed to help users explore career paths using AI-assisted guidance and modern UI features.

## Features Implemented
- 🔍 Search functionality for quick navigation
- 🎯 Filter buttons (All, Completed, Pending)
- 🔔 Notification system (UI-based alerts)
- 💬 Chat interface with simulated AI responses
- 🧭 Clean navigation with structured layout

## AI-Assisted Development
The application was enhanced using AI-assisted UI development (Lovable), where advanced navigation features such as search, filtering, and notifications were implemented.

## Tech Stack
- Lovable AI (UI Development)
- React-based components
- Modern responsive design

## Project Purpose
This project was developed as part of an academic assignment to demonstrate implementation of advanced navigation features and user experience improvements.
